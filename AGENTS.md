# Project Agents

## Documentation Automation Agent (Codex)

**Role**
- Serve as the primary automation assistant for documentation workflows.
- Maintain MkDocs configuration, structure, and supporting assets.

**Responsibilities**
- Translate requirements into MkDocs-compatible documentation and assets (Markdown, PlantUML, Kroki diagrams).
- Keep AGENTS.md and project-level meta-documentation current.
- Initialize and evolve the MkDocs project structure, ensuring docs build cleanly.
- Provide reproducible instructions for running and contributing to the documentation site.
- Track historical context for documentation updates to support transparency and reviews.
